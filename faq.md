# FAQ

I have summarized a few of the most frequently asked questions.

---

> *Installation failed, waypoints are missing, I don't know how to install it.*

There are no issues with the mod code.
This mod is based on D2RMM; while these issues were rare in the past, it seems quite a few users have been experiencing installation failures since the RotW update.
Although the exact cause has not been identified, it appears that these problems occur when D2RMM's copying process is not applied correctly or when the tool fails to modify the .txt files.
I recommend trying the following steps:

* Right-click d2rmm.exe, select "Run as administrator," and try reinstalling the mod.
* When you download the mod and extract the archive, a folder will be created. Copy that folder into D2RMM's "mods" folder. While the path may vary for everyone, it typically looks like this: D2RMM-169-1-8-0-1762645661\D2RMM 1.8.0\mods. (Do not copy the individual files inside the folder, such as mod.js or mod.json, into "MyMod" or move them around. You must not change the location of the files within the extracted folder. Please copy the entire folder exactly as it was after extraction. I will say this once more: copy the extracted folder itself, not the contents inside it.)
* I've also noticed cases where leftover junk files after the RotW release caused the game to load incorrect data. Therefore, I recommend a clean reinstallation of D2R for those experiencing similar issues.

> *﻿Error: BridgeAPI.openStorage: Failed to open CASC storage: CascLib error code 111: ERROR_BUFFER_OVERFLOW: The file name is too long.*

* It's not mod problem, this appears to be a compatibility issue between D2RMM and Steam users. If you download D2RMM 1.9 version, it will be fine.

---

> *﻿﻿Is it okay for my character if I turn the waypoint feature off after using it, or if I play normally at first and enable the waypoint feature later?*

* Every feature can be selected individually, and of course, it is perfectly safe. Even if you use all the features and then turn them off later, it will not cause any damage to your character, so you don't have to worry.

---

> *﻿﻿﻿Terror Zones are not activating.*

* Normal Terror Zones should function correctly. If they are not working, there are two likely scenarios. The first is an installation error as mentioned above, and the second is a conflict with other Terror Zone modification mods.
However, in the current version (v5.2), even if you use other Terror Zone mods simultaneously, most should work without issues depending on the mod load order; it's just that the Terror Zone color indicators for the added waypoints might not appear. This is something that would need to be fixed within that specific Terror Zone mod, so there is unfortunately nothing I can do about it on my end

---

> *﻿I am using other mods; can I use this one alongside them? Is it safe?*

* As stated on the mod page, in most cases, there are no issues if you place the QoL mod at the very bottom of your mod list before installing. However, a very small number of users have reported experiencing crashes when using it alongside certain mercenary-related mods. Additionally, compatibility with large-scale mods, including some custom map mods, varies depending on the specific mod. For example, Emuhel’s Multiboss mod was previously incompatible with QoL, but I have added a compatibility file for it today. Also, at the request of Korean users, a compatibility file for Reimagined has already been added.
Another mod I am interested in is Eastern Sun, but since the RotW compatibility update has not been released yet, I may add a compatibility file later once that update is complete. However, above all, my main focus is the vanilla experience, so I cannot guarantee compatibility with every single mod.

---

> *Isn't this a cheat?﻿*

* Whether this is a "cheat" depends on your perspective. Features like inventory expansion, rune stacking, or even changing the Cow Level recipe could be viewed that way.
The primary goal of this mod is Quality of Life (QoL). While some features like Anya's location or level skips may feel like cheats, every option is individually selectable. This allows you to enable only the features that align with your personal standards and playstyle.
The same applies to level skips. If a high-level player opens a portal for you in multiplayer, is that a cheat? Ultimately, you can enable only the features that fit your own standards and playstyle.
