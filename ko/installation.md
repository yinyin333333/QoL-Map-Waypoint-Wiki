첨부된 영상은 D2RMM과 오버홀 모드인 Reimagined, 그리고 QoL Map Waypoint를 설치하는 과정을 보여줍니다.
아래 설명된 내용과 동일하지만, 영상을 통해 더 자세한 과정을 확인하실 수 있습니다.

---

<a href="https://github.com/olegbl/d2rmm/releases/tag/v1.9.0" target="_blank" rel="noopener noreferrer">D2RMM 다운로드링크</a>

<img src="https://i.imgur.com/E6CQldX.png" />

다운로드 링크에서 D2RMM 1.9 압축 파일을 받아 원하는 위치에 압축을 풀어주세요.

<img src="https://i.imgur.com/piF8hRQ.png" />

압축을 푼 폴더에 있는 D2RMM.exe를 실행합니다.

<img src="https://i.imgur.com/as4IlOj.png" />

이제 D2RMM이 실행되었습니다. 다음으로 QoL Map Waypoint 모드를 다운로드해 봅시다.

<a href="https://www.nexusmods.com/diablo2resurrected/mods/906?tab=files" target="_blank" rel="noopener noreferrer">QoL Map Waypoint 다운로드링크</a>

<img src="https://i.imgur.com/4XI9mgQ.png" />

Manual download를 클릭하여 압축 파일을 다운로드합니다. 다운로드한 모드 압축 파일은 압축을 풀 필요가 없습니다.

<img src="https://i.imgur.com/AAIzEuO.gif" />

그다음, 다운로드한 QoL Map Waypoint 압축 파일을 앞서 열어둔 D2RMM 창으로 드래그 앤 드롭하기만 하면 됩니다.
오른쪽에 있는 설정(톱니바퀴 아이콘)을 클릭하여 원하는 옵션을 조절한 뒤, D2RMM 하단의 Install Mods를 클릭하면 설치가 완료됩니다.
이제 Run D2R을 눌러 게임을 즐기시면 됩니다.
