# Extra feature
Purely optional extra convenience features. Disabled by default.

---

<img src="https://i.imgur.com/8iycFfR.png" />
<img src="https://i.imgur.com/dNcKhJT.png" />

* Removes the need for keys when opening locked chests.

* The screen no longer darkens upon entering a Terror Zone. Additionally, even if the Terror Zone environment is set to day, there is usually a lingering transition effect where the screen momentarily dims and then brightens; this effect has also been disabled.

* Allows all non-passive, non-aura skills for playable classes to be assigned to both left and right mouse buttons.

* Allows all non-passive, non-aura skills can use in Town

* Prevent Flying Monster out of path (Act 2, Act4, Act5)

* Remove Dark screen edge

* You can skip the intro videos when launching D2R

* You can Skips the cinematic videos that play during the first travel between acts after completing a quest

* Previously, you had to click 'Play' on the main screen and then select the difficulty. However, when this option is enabled, buttons for Normal, Nightmare, and Hell difficulties will appear, allowing you to jump directly into each difficulty.(However, consoles and gamepads are not supported. Also, please note that if you click on Nightmare difficulty before you have actually unlocked it, a "Failed to Join Game" error will appear. Please keep this in mind.)

* You can adjust monster density as you wish

* You can adjust the experience gain scale. You can lower it for slower leveling or increase it for faster leveling
