# Known Issues

---

* Compatibility with some mods that change NPC locations (for example, moving Cain) is not guaranteed. Therefore, this mod provides those convenience features as well.
* Many updates have been made, and it is currently highly optimized. However, compatibility with every single mod cannot be guaranteed. At this time, compatibility with Vanilla, Reimagined, and Multibose is confirmed. I also plan to provide updates for overhaul mods like Eastern Sun and BTDiablo once their ROTW updates are released.
