# Installation

---

The QoL Map Waypoint mod is built for D2RMM.
However, the installation and application process is very straightforward.

---

<a href="https://github.com/olegbl/d2rmm/releases/tag/v1.9.0" target="_blank" rel="noopener noreferrer">D2RMM Download Link</a>

<img src="https://i.imgur.com/E6CQldX.png" />

Download D2RMM 1.9 zip from the download link and extract it to your preferred location.

<img src="https://i.imgur.com/piF8hRQ.png" />

There is a D2RMM.exe in the extracted folder. Run it.

<img src="https://i.imgur.com/as4IlOj.png" />

D2RMM is now running. Now, let's download the QoL Map Waypoint mod.

<a href="https://www.nexusmods.com/diablo2resurrected/mods/906?tab=files" target="_blank" rel="noopener noreferrer">QoL Map Waypoint Download Link</a>

<img src="https://i.imgur.com/4XI9mgQ.png" />

Click Manual download and download the zip file. You do not need to extract the downloaded mod zip file.

<img src="https://i.imgur.com/AAIzEuO.gif" />

Then, simply drag and drop the downloaded QoL Map Waypoint zip file into the D2RMM window you opened earlier. Click the Settings gear icon on the right to adjust your options, and then click Install Mods at the bottom of D2RMM to complete the installation. After that, just click Run D2R to play the game.
